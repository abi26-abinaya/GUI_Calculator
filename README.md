# 🧮 Simple GUI Calculator Using Python Tkinter

A beginner-friendly Calculator application built using **Python** and **Tkinter**. This project performs basic arithmetic operations through a simple graphical user interface (GUI).

> This project is created for learning Python GUI development using the Tkinter library.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🧹 Clear Screen
- 📱 Simple and Easy-to-use GUI
- ⚡ Instant Calculation

---

## 🛠️ Technologies Used

- Python 3.x
- Tkinter (Built-in Python GUI Library)
